Instructions
============
There are eight tabs which provide different types of rolls: coin toss, four sided die, six sided die, eight sided die, ten sided die, twelve sided die, twenty sided die, and an ability score generator based on the "roll four six sided die and keep the top three for each of six stats" method
Left and right keys move between tabs
up and down keys move through the roll history of the currently selected tab
The Enter/Return key makes a roll based on the currently selected tab
Space bar reads back the currently selected roll
The c key copies the currently selected roll to the clip board
